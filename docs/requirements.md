# 项目需求与验收标准

MVP 功能：
- 用户注册 / 登录（Firebase Auth 或同等）
- 3D 大厅场景（可切换样式）与 3D 宠物基础交互（抚摸/喂食/洗澡）
- 1:1 实时视频通话（移动 ↔ Windows）且支持本地音乐播放与通话并存
- 空间管家 Agent（文本命令，能下发结构化动作如换墙纸/播放音乐）
- 情侣相册、聊天与亲密度显示

验收标准示例：
- 两个账号在不同设备上可完成 1:1 视频通话（测试：Android + Windows）。
- 宠物的三项属性（心情/健康/饥饿）由交互改变并在另一端同步。
- Agent 能返回可执行 action JSON，App 根据 action 执行并回报结果。
